Design rule

Conformance is enforced in code, not prompts. The LLM decides what to change; the deterministic layer guarantees the result is a valid bundle.

Source of truth

Only the markdown is authoritative. Everything under .index/ and .traces/ is derived, disposable, and rebuilt from content hashes.

Models are plumbing

Any OpenAI-compatible endpoint works. Model discovery reads /v1/models and prefers the loaded model, so llama-swap works unmodified.

Why it matters locally

A 3B-active model can write mediocre prose, but it structurally cannot corrupt the bundle format — validation, indexing, and logging never pass through the model.

Graph health

memory_maintain runs a deterministic lint — orphans, broken links, undated Events — then lets the agent wire the fixes.

Hand edits welcome

Edit any markdown file directly: content hashes (never mtimes) notice the change and derived state catches up on the next touch.

System of record

The current snapshot stays clean for retrieval; the history stays queryable for questions about the past. Both, not either.

Legacy backfill

An existing log.md is backfilled into events on first touch — older bundles upgrade themselves.

Git as backup

GIT_AUTOCOMMIT commits each mutation; .events.jsonl diffs as exactly one added line. Add a remote push for real backup.

The budget rule

Spend deliberation where errors persist (writes, maintenance); spend infrastructure where errors are recoverable (reads). A bad read costs a retry; a bad write costs graph integrity.

Write protocol

Search first → enrich over create → link both ways → supersede contradictions → log the change. Enforced by prompt; mopped up by the deterministic lint.

Every run leaves a trace

Searches, reads, and writes record to .traces/ with the model chain — replayable as numbered hops on the graph view.

The pattern behind every fix: keep the current snapshot cheap to assemble, and make growth degrade gracefully — never silently. Nothing above changes what the librarian sees in its tools; it changes how much it costs to show it.

Thinking on the Spark

Escape hatches
